# dockerapps
run apps with docker

## 1. comfyui

**windows**
```pwsh
# setup and start (for first time)
./bin/comfyui.ps1 up -d

# stop
./bin/comfyui.ps1 stop

# start
./bin/comfyui.ps1 start
```

**linux**
```sh
# setup and start (for first time)
./bin/comfyui.sh up -d

# stop
./bin/comfyui.sh stop

# start
./bin/comfyui.sh start
```

> Open in browser:  http://localhost:8188
